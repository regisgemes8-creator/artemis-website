// INSTALLATION

Up in 60 seconds.

01

Install Fabric 1.21.1

Download the Fabric installer, select Minecraft 1.21.1, click Install. Launch Minecraft once with the new Fabric profile to generate the mods folder.

fabric installer →
02

Drop the .jar

Download artemis-1.0.0.jar below and place it in .minecraft/mods/. No other dependencies or Fabric API required.

download artemis →
03

Launch & Configure

Launch with the Fabric 1.21.1 profile. Press Right Shift to open ClickGUI. Toggle modules, set keybinds, and join DonutSMP.

right shift → clickgui
